Necessary Steps:

If you are using visual studdio to run app.js
  1. Download extension: Live Server (https://marketplace.visualstudio.com/items?itemName=ritwickdey.LiveServer)
  2. Click "Go Live" on the bottom right corner of your visual studio
  3. A window should pop out with the map on it
  
Functions:
1. Pan to current location
2. Display study areas based on filters applied
3. Clicking on any marker will only be for NTU location
